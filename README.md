# 🎓 Student Event Portal

A simple web-based Student Event Portal built using HTML, CSS, and JavaScript.  
It allows students to view and register for college events easily.

---

## 🚀 Features

- View upcoming college events  
- Event registration form  
- Simple and clean UI  
- Easy navigation between pages  

---

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- JavaScript
